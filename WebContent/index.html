Introductory Web App

The main purpose of this app is to practice deploying apps on a server and accessing the various components. Later tutorial sections will discuss the actual code.
  • Hello.html
  • Hello.jsp
  • HelloWorld servlet (plain text output). Uses shortcut invoker servlet address that requires no entries in web.xml. This shortcut URL is for practice only.
  • HelloWorld servlet. Uses custom address in web.xml. All real projects should use custom addresses.
  • HelloServlet (HTML output). Uses shortcut invoker servlet address.
  • HelloServlet (HTML output). Uses custom address in web.xml.
  • HelloServlet2. Uses packages and invoker servlet address.
  • HelloServlet2. Uses packages and custom address in web.xml. All real projects should use packages for all classes and custom addresses for all servlets.
  • HelloServlet3. Uses helper classes, packages, and invoker servlet address.
  • HelloServlet3. Uses helper classes, packages, and custom address in web.xml.
